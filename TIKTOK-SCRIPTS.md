## Filming Tips

| Do | Don't |
|----|-------|
| Film in van, on job, real environment | Studio/office |
| Specific real numbers | Vague "save money" |
| Calm, direct (mate to mate) | Hyped/salesy |
| Speak slower, pause for effect | Speak fast |
| Show face, work context first | Fancy transitions |
