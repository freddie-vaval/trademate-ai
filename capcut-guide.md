## CapCut Assembly Steps

### Step 1: Create New Project
1. Open CapCut
2. Create new project
3. Set aspect ratio: **9:16** (vertical for TikTok/Reels)

### Step 2: Add Images + Timing
| Time | Image | Duration |
|------|-------|----------|
| 0:00-0:05 | 01-hook-phone.png | 5s |
| 0:05-0:15 | 02-problem-tradesman.png | 10s |
| 0:15-0:30 | 03-solution-ai.png | 15s |
| 0:30-0:35 | 04-feature-quotes.png | 5s |
| 0:35-0:40 | 05-feature-schedule.png | 5s |
| 0:40-0:45 | 06-feature-finance.png | 5s |
| 0:50-0:55 | 03-solution-ai.png | 5s (reuse for CTA) |

### Step 3: Add Voiceovers
1. Import all `.m4a` files
2. Match audio to image timing above
3. Adjust image duration to match audio

### Step 4: Add Text Overlays
Add text on screen for each feature:
- 0:00 - "£800 LOST"
- 0:30 - "AI Phone Receptionist"
- 0:35 - "Quotes in Seconds"
- 0:40 - "Smart Scheduling"
- 0:45 - "Financial Tracking"
- 0:50 - "£2,400/month recovered"
- 0:55 - "TradeMateAI.co.uk"

### Step 5: Add Music
- Search "cinematic" or "tension" in CapCut music
- Or use free stock music

### Step 6: Export
- Export 16:9 (YouTube)
- Export 9:16 (TikTok/Reels)

## Pro Tips

1. **Transitions** - Use simple cuts, not fancy transitions
2. **Text** - Make it big, white with dark outline
3. **Pacing** - Let each moment breathe
4. **Hook** - First 3 seconds are everything
